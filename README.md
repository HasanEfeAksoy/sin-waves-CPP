# sin-waves-CPP
<b>sinus waves with SFML</b><br /><br />
<i>*you can change lambda, amplitude and speed.</i><br /><br />

<u><b>CONTROLS:</b></u><br /><br />

q => increase lambda<br />
w => reduce lambda<br />
e => increase -amplitude<br />
r => reduce -amplitude<br />
t => increase speed<br />
y => reduce speed<br />
a => reload values<br />
